Data Structure 2D (Input)

Data struktur ini merepresentasikan posisi dan properti paku (nail) pada grid 2D:

  • nails: Object berisi data paku dengan key koordinat "x,y"
  • height: Tinggi paku dalam mm (1-100)
  • body_width: Lebar badan paku (1=thin, 2=medium, 3=thick)
  • head_width: Lebar kepala paku (1=small, 2=medium, 3=large)
  • gridWidth & gridHeight: Ukuran grid
  • marginBetweenNails: Jarak antar paku dalam mm

Proses Konversi 2D ke 3D

1

Parsing Data Structure

Data JSON 2D diparsing untuk mengekstrak informasi posisi (x,y), tinggi (height), dan dimensi paku (body_width, head_width).

2

Konversi Koordinat

Koordinat grid 2D (x,y) dikonversi ke koordinat 3D Three.js (x,y,z) dengan perhitungan:

posX = (x - (gridWidth - 1) / 2) × marginBetweenNails
posZ = (y - (gridHeight - 1) / 2) × marginBetweenNails
3

Instancing Geometry

Menggunakan THREE.InstancedMesh untuk render efisien ribuan paku dengan 2 komponen:

  • Nail Shaft: CylinderGeometry dengan height dinamis
  • Nail Head: CylinderGeometry dengan radius lebih besar
4

Transformasi Matrix

Setiap instance paku ditransformasi menggunakan Matrix4 untuk positioning dan scaling:

  • Scale Y: Tinggi paku (height)
  • Scale X & Z: Lebar paku (body_width)
  • Position: Lokasi berdasarkan grid
